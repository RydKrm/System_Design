
# 🟢 LEVEL 1 — Visual Foundations (The Roots)

These are timeless principles. Designers have followed these long before mobile apps existed.

### 1. Design Principles (Core Rules)

- Visual hierarchy
    
- Contrast
    
- Alignment
    
- Repetition
    
- Proximity
    
- Balance (symmetry vs asymmetry)
    
- White space (negative space)
    
- Consistency
    

👉 If you deeply understand these, your UI will already look 10x better.

---

### 2. Color Theory (Real Understanding, Not Random Palettes)

- Color wheel basics
    
- Complementary / analogous colors
    
- Warm vs cool colors
    
- Saturation & brightness
    
- Emotional meaning of colors
    
- Accessibility contrast ratios (WCAG basics)
    
- Dark mode color adaptation
    

Practice: Recreate palettes used by apps like:

- Spotify
    
- WhatsApp
    
- Instagram
    

Ask yourself: Why these colors? Why this shade?

---

### 3. Typography (Very Underrated Skill)

- Font anatomy
    
- Serif vs Sans Serif
    
- Font pairing
    
- Line height (leading)
    
- Letter spacing
    
- Font weight hierarchy
    
- Mobile readability principles
    

Study fonts used in:

- Apple (SF Pro)
    
- Google (Roboto)
    

Typography alone can make a UI look premium.
