# Overview

## What is React Navigation?

**React Navigation** is the most popular navigation library for React Native. It provides a way to navigate between screens, manage navigation history, and handle deep linking.

### Why React Navigation?

1. **JavaScript-based** - Pure JavaScript, no native code needed
2. **Customizable** - Highly flexible and configurable
3. **Cross-platform** - Works on iOS, Android, and Web
4. **Active development** - Well-maintained by the community
5. **TypeScript support** - Full type safety

### Alternatives

- **React Native Navigation** (by Wix) - Native navigation, better performance
- **React Router Native** - Web-like routing
- **Expo Router** - File-based routing (built on React Navigation)

# Navigation Types

React Navigation provides several navigator types:

## 1. Stack Navigator

**Use Case**: Push/pop screens like a stack (modal navigation) **Example**: Login → Home → Profile → Settings

## 2. Tab Navigator (Bottom Tabs)

**Use Case**: Primary navigation with tabs at the bottom **Example**: Instagram (Home, Search, Reels, Shop, Profile)

## 3. Material Top Tabs

**Use Case**: Swipeable tabs at the top **Example**: WhatsApp (Chats, Status, Calls)

## 4. Drawer Navigator

**Use Case**: Side menu navigation **Example**: Gmail (hamburger menu with folders)

## 5. Native Stack Navigator

**Use Case**: Native navigation performance **Example**: Large apps needing better performance

# Stack Navigator

## Complete Implementation

```typescript
import React from 'react';
import { NavigationContainer } from '@react-navigation/native';
import { createStackNavigator } from '@react-navigation/stack';
import HomeScreen from './screens/HomeScreen';
import DetailsScreen from './screens/DetailsScreen';

// Define types for type-safe navigation
type RootStackParamList = {
  Home: undefined;
  Details: { itemId: number; otherParam?: string };
};

const Stack = createStackNavigator<RootStackParamList>();

function App() {
  return (
    <NavigationContainer>
      <Stack.Navigator
        initialRouteName="Home"
        screenOptions={{
          headerStyle: { backgroundColor: '#f4511e' },
          headerTintColor: '#fff',
          headerTitleStyle: { fontWeight: 'bold' },
        }}
      >
        <Stack.Screen 
          name="Home" 
          component={HomeScreen}
          options={{ title: 'My Home' }}
        />
        <Stack.Screen 
          name="Details" 
          component={DetailsScreen}
          options={({ route }) => ({ 
            title: `Item ${route.params.itemId}` 
          })}
        />
      </Stack.Navigator>
    </NavigationContainer>
  );
}
```

## Stack.Navigator Properties

### `initialRouteName`

```typescript
<Stack.Navigator initialRouteName="Home">
```

- **Type**: `string`
- **Purpose**: First screen to render when navigator loads
- **Default**: First screen in the stack
- **Example**: Set to "Login" for auth flow

### `screenOptions`

```typescript
<Stack.Navigator
  screenOptions={{
    headerStyle: { backgroundColor: '#f4511e' },
    headerTintColor: '#fff',
    headerTitleStyle: { fontWeight: 'bold' },
    headerShown: true,
  }}
>
```

- **Type**: `object` or `function`
- **Purpose**: Default options for all screens
- **Can be overridden**: By individual screen options
- **Function form**: Access navigation and route
    
    ```typescript
    screenOptions={({ navigation, route }) => ({  headerTitle: route.name,})}
    ```
    

### `detachInactiveScreens`

```typescript
<Stack.Navigator detachInactiveScreens={true}>
```

- **Type**: `boolean`
- **Purpose**: Unmount inactive screens to save memory
- **Default**: `true` on Android, `false` on iOS
- **Use when**: Large apps with many screens

### `screenListeners`

```typescript
<Stack.Navigator
  screenListeners={{
    focus: (e) => {
      console.log('Screen focused:', e.target);
    },
    blur: (e) => {
      console.log('Screen blurred:', e.target);
    },
  }}
>
```

- **Type**: `object`
- **Purpose**: Listen to navigation events on all screens
- **Events**: `focus`, `blur`, `state`, `beforeRemove`

# Tab Navigator

## Bottom Tabs Implementation

```typescript
import { createBottomTabNavigator } from '@react-navigation/bottom-tabs';
import { Ionicons } from '@expo/vector-icons';

const Tab = createBottomTabNavigator();

function MyTabs() {
  return (
    <Tab.Navigator
      initialRouteName="Home"
      screenOptions={({ route }) => ({
        tabBarIcon: ({ focused, color, size }) => {
          let iconName;
          if (route.name === 'Home') {
            iconName = focused ? 'home' : 'home-outline';
          } else if (route.name === 'Settings') {
            iconName = focused ? 'settings' : 'settings-outline';
          }
          return <Ionicons name={iconName} size={size} color={color} />;
        },
        tabBarActiveTintColor: 'tomato',
        tabBarInactiveTintColor: 'gray',
      })}
    >
      <Tab.Screen name="Home" component={HomeScreen} />
      <Tab.Screen name="Settings" component={SettingsScreen} />
    </Tab.Navigator>
  );
}
```


## Navigation Prop

## Navigation Methods

### `navigate`

```typescript
navigation.navigate('Details', { itemId: 42 });
```

- **Purpose**: Navigate to a screen
- **Params**: Screen name and optional parameters
- **Behavior**:
    - If screen exists in stack, goes to it
    - If not, pushes new screen

### `push`

```typescript
navigation.push('Details', { itemId: 42 });
```

- **Purpose**: Push new screen on stack
- **Difference from navigate**: Always adds new screen
- **Use case**: Multiple instances of same screen

### `goBack`

```typescript
navigation.goBack();
```

- **Purpose**: Go to previous screen
- **Params**: None
- **Behavior**: Pops current screen from stack

### `pop`

```typescript
navigation.pop();
navigation.pop(2); // Go back 2 screens
```

- **Purpose**: Pop one or more screens
- **Params**: Optional number of screens to pop
- **Default**: 1

### `popToTop`

```typescript
navigation.popToTop();
```

- **Purpose**: Go to first screen in stack
- **Params**: None
- **Use case**: Reset to home screen

### `replace`

```typescript
navigation.replace('Profile');
```

- **Purpose**: Replace current screen
- **Behavior**: Current screen removed from stack
- **Use case**: Login → Home (prevent going back)

### `reset`

```typescript
navigation.reset({
  index: 0,
  routes: [{ name: 'Home' }],
});
```

- **Purpose**: Reset navigation state
- **Params**: New navigation state
- **Use case**: Logout, reset app flow

### `setParams`

```typescript
navigation.setParams({ query: 'updated' });
```

- **Purpose**: Update screen params
- **Params**: Object with params to update
- **Use case**: Update search query, filters

### `setOptions`

```typescript
navigation.setOptions({
  title: 'Updated Title',
  headerRight: () => <Button title="Save" />,
});
```

- **Purpose**: Update screen options dynamically
- **Params**: Options object
- **Use case**: Dynamic headers, conditional buttons

### `openDrawer`

```typescript
navigation.openDrawer();
```

- **Purpose**: Open drawer
- **Drawer Navigator only**

### `closeDrawer`

```typescript
navigation.closeDrawer();
```

- **Purpose**: Close drawer
- **Drawer Navigator only**

### `toggleDrawer`

```typescript
navigation.toggleDrawer();
```

- **Purpose**: Toggle drawer open/closed
- **Drawer Navigator only**

### `jumpTo`

```typescript
navigation.jumpTo('Settings');
```

- **Purpose**: Jump to a tab/drawer screen
- **Tab/Drawer Navigator only**
- **Difference from navigate**: Doesn't add to history

### `isFocused`

```typescript
const isFocused = navigation.isFocused();
```

- **Purpose**: Check if screen is focused
- **Returns**: `boolean`
- **Use case**: Conditional logic based on focus

### `canGoBack`

```typescript
const canGoBack = navigation.canGoBack();
```

- **Purpose**: Check if can go back
- **Returns**: `boolean`
- **Use case**: Show/hide back button

### `getParent`

```typescript
const parentNavigation = navigation.getParent();
```

- **Purpose**: Get parent navigator
- **Returns**: Parent navigation object
- **Use case**: Access parent navigator methods

### `getState`

```typescript
const state = navigation.getState();
```

- **Purpose**: Get current navigation state
- **Returns**: Navigation state object
- **Use case**: Custom navigation logic

### `addListener`

```typescript
const unsubscribe = navigation.addListener('focus', () => {
  console.log('Screen focused');
});

// Later: unsubscribe()
```

- **Purpose**: Listen to navigation events
- **Events**: `focus`, `blur`, `state`, `beforeRemove`
- **Returns**: Unsubscribe function

### `removeListener`

```typescript
navigation.removeListener('focus', handler);
```

- **Purpose**: Remove event listener
- **Usually use**: Unsubscribe function instead

### `dispatch`

```typescript
import { CommonActions } from '@react-navigation/native';

navigation.dispatch(
  CommonActions.navigate({
    name: 'Profile',
    params: { userId: 'jane' },
  })
);
```

- **Purpose**: Dispatch navigation action
- **Advanced**: For custom navigation logic

```typescript

import { createNavigationContainerRef } from '@react-navigation/native';

export const navigationRef = createNavigationContainerRef();

export function navigate(name: string, params?: object) {
  if (navigationRef.isReady()) {
    navigationRef.navigate(name, params);
  }
}

import * as NavigationService from './navigationService';
NavigationService.navigate('Home');
```
