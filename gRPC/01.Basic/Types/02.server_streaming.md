## **8. Pros and Cons**

**Pros:**

* Efficient for large/multiple results.
* Client can process data as it arrives.

**Cons:**

* Slightly more complex than Unary RPC.
* Client must handle stream events (`data`, `end`).
* Not ideal if you truly only need one response.
